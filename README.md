# base.c

Base number conversion utility.

A little project to learn about libc standard libraries, make and
libraries made in C.

# Using base.c

Usage: `base [OPTIONS] NUMBER...`

# Build, install, test...

system dependencies:
- gnu make
- C compiler (gcc, clang, zig... )

```bash
# Build project
make

# Test
make test

# Install on system
make install

# Uninstall from system
make uninstall
```
