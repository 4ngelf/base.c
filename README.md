# base.c

Base number conversion utility.

A little project to learn about libc standard libraries and
how to make projects in C.

## Using base.c

Usage: `base [OPTIONS] NUMBER...`

## Build, install, test...

System dependencies:
- gnu make
- C compiler (gcc, clang, zig... )

```bash
# Build project
make

# Test project
make test

# Install on system
make install

# Uninstall from system
make uninstall
```
